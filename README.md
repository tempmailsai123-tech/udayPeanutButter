# Uday Peanut Butter

Premium peanut butter brand website built with React, TypeScript, Vite, and Tailwind CSS.

## Live site

After GitHub Pages is enabled, the site will be available at:

**https://tempmailsai123-tech.github.io/udayPeanutButter/**

## Run locally

```bash
git clone https://github.com/tempmailsai123-tech/udayPeanutButter.git
cd udayPeanutButter
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

Pushes to the `main` branch automatically build and deploy via GitHub Actions.

To enable the live site:

1. Open the repo on GitHub → **Settings** → **Pages**
2. Under **Build and deployment**, set **Source** to **GitHub Actions**
3. Push to `main` (or re-run the **Deploy to GitHub Pages** workflow)
